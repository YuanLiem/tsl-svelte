# tsl-svelte

SvelteKit recipe finder using dummyjson.com

- SvelteKit & Svelte 5
- Tailwind CSS
- shadcn-svelte
- vitest
- playwright

## Setup

Prerequisits

- Node LTS
- pnpm `npm install -g pnpm`

1. Clone the repository
   `git clone https://github.com/YuanLiem/tsl-svelte`

2. Install dependencies
   `pnpm install`

3. Start the development server
   `pnpm run dev`
